== BC3.1 : Installation et Configuration

*Auteur :* [Nom de l'étudiant]

L'infrastructure repose sur un déploiement Docker multi-conteneurs. Voici la procédure de déploiement :

[source,bash]
----
docker compose up --build -d
----

*Preuve :*
